# Unified Mentor Internship Projects

This folder contains the projects completed as part of the **Unified Mentor Data Analyst Internship**.

These projects demonstrate practical experience in data analysis, visualization, and business insight generation using tools such as Excel, SQL, Python, and Power BI.

## Projects Included

### E-commerce Sales Analysis
Sales analysis using Excel to identify product performance, revenue trends, and customer purchasing patterns.

### IBM HR Employee Attrition Analysis
Analysis of employee attrition using Excel, SQL, and Power BI to identify workforce trends and key HR insights.

### Netflix Data Analysis
Data exploration of Netflix content using Excel and Python (Pandas, Matplotlib) to analyze content distribution and genre trends.

### Supermart Sales Analysis
Retail sales analysis using Excel and Power BI to evaluate regional performance and profitability insights.
